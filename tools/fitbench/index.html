FitBench

least-squares curve fitting · runs entirely offline

paste data → pick a model → fit. parameters come with uncertainties.

Data

Two or three columns: x  y  [y-error]. Space, comma or tab separated. Lines starting with # are ignored.

Model

Use x and single-letter parameters. Functions: exp, ln, log, sin, cos, sqrt, abs, erf, gauss(x,μ,σ)…

tip: Ctrl+Enter in the data box also fits
